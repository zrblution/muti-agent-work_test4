# Multi-Agent MLLM Hallucination Mitigation

本项目用于构建一个基于多 Agent 协作机制的多模态大模型幻觉消除框架。

## 项目目标

多模态大模型在图像理解、视觉问答、图文推理等任务中容易产生与视觉证据不一致的回答。本项目计划通过多 Agent 的协作、校验、反思和裁决机制，降低模型输出中的事实性错误和视觉幻觉问题。

## 核心思路

- 使用多个 Agent 分别承担感知、推理、检索、验证和裁决等角色。
- 引入跨 Agent 的一致性检查，识别可能的幻觉内容。
- 结合视觉证据、文本上下文和外部知识，对模型回答进行验证和修正。
- 构建可扩展的实验框架，支持不同多模态大模型、数据集和评估指标。
